# Poke Tools

Application React/Vite de consultation Pokemon basee sur l'API Tyradex.

## Source de donnees

- Liste Pokemon: `https://tyradex.app/api/v1/pokemon`
- Detail Pokemon: `https://tyradex.app/api/v1/pokemon/:id`

Le projet utilise Tyradex comme source principale et mappe les reponses de l'API vers les composants de l'interface.

## Scripts

- `npm run dev` lance le serveur de developpement Vite
- `npm run build` genere le build de production
- `npm run lint` lance ESLint
